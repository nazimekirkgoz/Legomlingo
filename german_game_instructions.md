# German Article Learning Game

Ein interaktives Lernspiel für deutsche Artikel (Der, Die, Das) mit LEGO Boost.

## Farben & Artikel
*   **Rot (RED)** = Die
*   **Blau (BLUE)** = Der
*   **Grün (GREEN)** = Das

## Spielablauf
1.  **Vorbereitung**: 5 Wörter werden angezeigt. Merke sie dir!
2.  **Frage**: Eine Definition erscheint. Du hast 5 Sekunden Zeit zu überlegen.
3.  **Antwort**: Zeige die passende Farbkarte (LEGO Stein) dem Sensor.
    *   Richtig: Das Licht wird Grün.
    *   Falsch: Das Licht wird Rot. Versuche es nochmal!

## Starten
Führe den folgenden Befehl im Terminal aus:
```bash
python german_learning.py
```

## Tastatur-Steuerung (zum Testen ohne LEGO)
*   **R**: Simuliert Rot (Die)
*   **B**: Simuliert Blau (Der)
*   **G**: Simuliert Grün (Das)
*   **ESC**: Beenden
